  <h2>🛠 Kurulum</h2>
  <pre>
  1. Bu repoyu klonlayın:
     git clone https://github.com/kullaniciadi/nodejs-ogrenme-serisi.git

  2. Proje klasörüne girin:
     cd nodejs-ogrenme-serisi

  3. Her klasör içinde bağımlılıkları yükleyin:
     npm install

  4. Uygulamayı başlatın:
     npm start
  </pre>
